Pre-Bootcamp Coding Class

An introduction to the basics of front-end web dev

About

the pre-bootcamp coding classes were started by MaiAda (emotional unicorn) on twtch. this was a response to the fact that so many of her friends were curious about whether bootcamps were the right fit for them. she realized how overwhelming it could be to even know where to start with learning how to code and deciding whether it was the right fit.

the purpose of this course is to familiarize newbie beginners to coding with the syntax and language of HTML, CSS and Javascript. the goal is for anybody coming into the class to feel comfortable writing basic code and confident enough to experiment and learn on their own.

you can participate live on twitch on thursdays at 6pm est. all vods are saved so you can go review them anytime you want.
