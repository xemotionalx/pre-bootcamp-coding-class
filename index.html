Pre-Bootcamp Coding Class

An introduction to the basics of front-end web dev

About

The Pre-Bootcamp coding classes were started by MaiAda (emotional unicorn) on Twitch. This was a response to the fact that so many of her friends were curious about whether bootcamps were the right fit for them. She realized how overwhelming it could be to even know where to start with learning how to code and deciding whether it was the right fit.

The purpose of this course is to familiarize total beginners to coding with the syntax and language of HTML, CSS and Javascript. The goal is for anybody coming into the class to feel comfortable writing basic code and confident enough to experiment and learn on their own.

You can participate live on Twitch on Thursdays at 6pm est. All VODs are saved so you can go review them anytime you want.
